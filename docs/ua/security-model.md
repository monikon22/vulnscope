# Модель безпеки

VulnScope призначений лише для авторизованого тестування. Він забезпечує область дії, обмежує швидкість запитів, використовує таймаути та постачає безпечні корисні навантаження. Він не реалізує брутфорс, підбір облікових даних, руйнівні SQL-навантаження, видалення файлів, виконання команд оболонки, забезпечення стійкості, підвищення привілеїв або автоматизовану експлуатацію.

Секрети, надані для автентифікованих сканувань, видаляються зі сховища та звітів. Результати є сигналами виявлення, а не доказом можливості експлуатації.
